# 🇨🇲 PETIT QUIZ pour fuir le NKANG

Un jeu de rapidité et de connaissances géographiques développé pour tester tes connaissances sur les drapeaux et les capitales du monde entier.

## 🎮 Le Concept
Le principe est simple :
- Un drapeau s'affiche aléatoirement.
- Tu dois saisir le **nom du pays** et sa **capitale**.
- **Règle d'or :** Si tu as juste, tu gagnes 1 point. Si tu te trompes, ton score retombe à 0 !
- Le jeu ne propose jamais deux fois le même drapeau durant une partie.

## 🛠️ Technologies utilisées
- **HTML5 / CSS3** : Pour une interface moderne et élégante dans les tons gris.
- **JavaScript (ES6+)** : Pour la logique du jeu et la gestion du score.
- **API RestCountries** : Pour récupérer les données de plus de 240 pays et territoires en temps réel.
